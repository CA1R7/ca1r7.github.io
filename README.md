# Installation
Using yarn :
```
yarn
```
Or NPM :

```
npm install
```

# Usage
Don't forget to install the dependencies first and then run the command :
```
yarn dev
```

## Requirements
#### Node.js

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

#### Yarn (required)
It supports `>=1.22.5`
